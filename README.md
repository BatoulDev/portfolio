# Editorial Full-Stack Portfolio

A modern, clean portfolio website with an editorial design system.

## Quick Start

### Prerequisites
- Node.js 18+

### Terminal 1 - Backend (API)
```bash
cd portfolio/backend
npm run dev
```
Server runs at http://localhost:5000

### Terminal 2 - Frontend (Next.js)
```bash
cd portfolio/frontend
npm run dev
```
Site runs at http://localhost:3000

## Pages
- **Home** - http://localhost:3000
- **About** - http://localhost:3000/about
- **Projects** - http://localhost:3000/projects (GitHub API)
- **Contact** - http://localhost:3000/contact
- **Login** - http://localhost:3000/login
- **Register** - http://localhost:3000/register

## Design System
- **Primary**: #E2B4B7 (Soft Pink)
- **Background**: #FFF9ED (Creamy White)
- **Text**: #1D1C15 (Soft Black)

## Tech Stack
- Frontend: Next.js 14, React, Tailwind CSS, Framer Motion
- Backend: Node.js, Express, JWT, bcryptjs
